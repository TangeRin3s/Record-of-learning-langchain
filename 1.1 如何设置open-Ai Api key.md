
## 1. 获取 API Key

去 OpenAI Dashboard → API Keys 页面
生成一个类似这样的 key：

```python
sk-xxxxxxxxxxxxxxxx
```

## 2. 在 Windows（Conda / Anaconda）中设置

### 临时（当前终端有效，推荐测试用）
```python
set OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxx
```
⚠️ 注意：

不能有引号

当前窗口有效，关掉就没了

### 永久（推荐）

Windows 搜索 → “编辑系统环境变量”

环境变量 → 用户变量 → 新建

变量名：
```python
OPENAI_API_KEY
```


变量值：
```python
sk-xxxxxxxxxxxxxxxx
```

确认 → 关闭所有终端 → 重新打开

## 3. 验证是否成功
```python
python -c "import os; print(os.getenv('OPENAI_API_KEY'))"
```
如果能打印出 sk-xxxx，说明成功 ✅

## ！！注意
设置后：

Jupyter kernel 必须重启

VSCode 必须重开窗口

Python 进程不会自动刷新环境变量
